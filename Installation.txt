1. Download and install git.
2. First clone the repository. Go to github, click the "clone or download" button and coppy the IP address.
3. Open the git command line. Use the "cd" command to navigate to the place you want to download the repo. The type 

	git clone https://github.com/nicholgroup/special-measure.git

4. Open matlab, and add the repo directory with all subfolders to your path.
5. To commit changes to the repo, open the git shell. To stage all changes, type

	git add -A

Then type

	git commit -m "message describing changes"

Finally, to upload your changes to the server, type

	git push origin master

6. Ordinarily, if you are making big changes, you should work in a separate branch, which can later be merged with the master. In this case, before you commit, you should make and commit to a new branch. That way, you can safely make changes without affecting the master.

See http://dont-be-afraid-to-commit.readthedocs.io/en/latest/git/commandlinegit.html for more details.
